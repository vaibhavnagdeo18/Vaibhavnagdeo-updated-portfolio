Welcome to my portfolio! This repository showcases my work as a software engineer, highlighting expertise in AI, backend systems, and modern web development with a focus on high-performance 3D animations.

## Tech Stack

### **Frontend & UI**
- **Core**: React, TypeScript, HTML5, CSS3, JavaScript
- **Animations**: GSAP (GreenSock Animation Platform), WebGL
- **3D Rendering**: Three.js, @react-three/fiber, @react-three/drei
- **Tools**: Vite, ESLint

### **Backend & AI**
- **Frameworks**: Node.js, Express.js
- **Database / ORM**: PostgreSQL, Prisma
- **AI Integration**: AWS Bedrock, Google Gemini, Multi-Agent Systems

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.x or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/vaibhavnagdeo18/Vaibhavnagdeo-updated-portfolio.git
    cd Vaibhavnagdeo-updated-portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run locally**:
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:5173`.

> [!IMPORTANT]
> This project uses GSAP Trial plugins. For production hosting, please ensure you have the appropriate GSAP license. For more information, visit [GSAP Installation Guide](https://gsap.com/docs/v3/Installation/).
